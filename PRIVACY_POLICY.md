# Privacy Policy for Time Waste Watcher

Last updated: March 23, 2026

## Summary
Time Waste Watcher helps users limit time spent on selected websites. The extension stores settings and time-usage data locally in the browser.

## What data the extension handles
The extension processes:
- website domains the user chooses to track
- time spent on tracked websites
- daily and hourly limits configured by the user

## How data is used
This data is used only to:
- show countdown timers on tracked sites
- block a tracked site after a configured limit is reached
- save the user's website list and time-limit preferences

## Data storage
All extension data is stored locally using Chrome extension storage on the user's device.

## Data sharing
Time Waste Watcher does not sell, transmit, or share user data with the developer or with third parties.

## Remote services
Time Waste Watcher does not use remote servers, analytics services, or advertising networks.

## Permissions
The extension requests browser access needed to:
- store settings and usage data
- detect the active tab on tracked websites
- apply timers and blocking UI on websites the user tracks

## User controls
Users can:
- add or remove tracked websites
- set limits to 0 to disable them
- remove the extension at any time
- clear browser extension storage to remove saved settings and usage data
